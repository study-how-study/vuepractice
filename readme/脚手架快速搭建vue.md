# vue脚手架

## 配置淘宝镜像
```
npm config set registry https://registry.npm.taobao.org
```

## 创建vue项目(最新）
```
npm install -g @vue/cli
vue create vue_demo
```
## 启动
```
npm install 
npm run serve 
npm run build
npm run lint
```
